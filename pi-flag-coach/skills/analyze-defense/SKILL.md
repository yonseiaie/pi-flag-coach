# Analyze Defense — 실시간 수비 분석 & 전략 힌트

전략 힌트 패널에 출력되는 결론을 만드는 스킬. **추측 금지**, 항상 수치 근거.

## 데이터 소스
- `read_opponent_record` 도구 (최근 3경기 집계) 또는 MCP `stats-db` 쿼리
- 현재 다운&거리, 야드 라인, 점수, 남은 시간 (UI가 turn 컨텍스트로 주입)

## 판단 규칙 (임계값)
1. **세이프티 깊이 ≥ 70%** → 딥 존이 비어 있음 → **Y 또는 F 딥 패스** 추천.
2. **플랫 수비 성공률 ≤ 50%** → 플랫이 약함 → **H 플랫 / Z 슬롯 퀵 패스** 추천.
3. **블리츠 빈도 ≥ 65%** → 빠른 릴리스 필요 → **핫 루트(Mesh Switch) + 3스텝 드롭** 추천.
4. 위 조건 2개 이상 동시 충족 시, 충돌하면 **다운&거리**로 우선순위 결정
   (3rd & long → 딥, 3rd & short → 플랫/슬롯).

## 출력 형식 (전략 힌트 패널)
```
💡 <한 문장 추천>
   근거: <지표 1>, <지표 2>
   추천 플레이: <PLAY 이름>
```
예) "Y 또는 F에게 딥 패스 — 세이프티 깊이 72%로 깊게 위치. 추천: Mesh Switch."

## 위임
다중 지표 교차 분석이 무거우면 `defense-analyst` 서브에이전트에 위임하고,
공격 콜 설계는 `offense-strategist` 서브에이전트와 병렬로 진행한다.
