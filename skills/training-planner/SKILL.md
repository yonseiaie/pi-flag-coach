# Training Planner — 맞춤형 훈련 세션 자동 생성

훈련 계획 섹션의 자동 생성을 담당한다.

## 입력
- 대상 선수/포지션 (없으면 로스터 전체)
- 목표 (예: 딥 패스 정확도, 블리츠 대응, 플랫 커버리지)
- 약점 신호: `analyze-defense` 결과 또는 선수 스탯의 부진 항목

## 절차
1. `get_player_stats` 로 대상 선수의 약점 지표를 식별한다.
   (예: Q의 INT, X의 drops, Z의 3rd-down 컨버전)
2. 약점 → 드릴 매핑 (아래 표)으로 60~90분 세션을 구성한다.
3. 워밍업 10분 → 포지션 드릴 → 팀 스크림 → 쿨다운 순서로 배치한다.
4. `create_training_session` 도구로 `training-plans/<날짜>-<목표>.md` 저장.
5. 저장 후 `background-notify` 가 "훈련 계획 생성 완료" 알림을 보낸다.

## 약점 → 드릴 매핑
| 약점 지표 | 드릴 |
|---------|-----|
| Q INT 과다 | 프리스냅 리드 + 타이밍 패스 래더 |
| WR drops | 핸즈 캐치 + 트래픽 캐치 |
| 3rd-down 부진 | 컨버전 시뮬레이션(존/맨 혼합) |
| 블리츠 약점 | 핫 루트 인식 + 3스텝 릴리스 |
| 딥 정확도 | 에어 야드 타깃팅 (15/25/35yd) |

## 출력
세션 제목, 총 시간, 블록별(시간·드릴·목표) 표, 코칭 포인트 3개.
